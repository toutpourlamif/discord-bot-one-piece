# One Piece Discord Bot

Un bot Discord de jeu dans l'univers One Piece : recrutement d'équipage, amélioration de navire, exploration, combat.

## Quickstart

_À compléter au premier code runnable._

```bash
pnpm install
pnpm dev
```
